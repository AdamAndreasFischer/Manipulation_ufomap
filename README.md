# This repo is a clone of https://github.com/UnknownFreeOccupied/ufomap
This repo is a clone of UFOMap which has an added manipulation package. 

# Install 
In `lib/map/apps/manipulation` there is a `README.MD` which briefly explains how to install the package. Nothing else needs to change in the package for it to work with the pointcloud package. Make sure to install the prerequisites in the readme before install.
